# 🌳 LeDOM — Comprendre le Document Object Model

> *"Le DOM, c'est le pont entre votre HTML et JavaScript. Maîtrisez-le, maîtrisez le web."*

---

## 🎯 À propos du projet

**LeDOM** est un exposé pédagogique complet sur le **Document Object Model (DOM)** — la représentation en mémoire d'une page web, manipulable en JavaScript.

Ce projet a été conçu pour comprendre en profondeur comment le navigateur interprète le HTML et comment JavaScript interagit dynamiquement avec la page.

## 🚀 Lancer le projet

1. **Cloner le dépôt**
   ```bash
   git clone https://github.com/2020stephane/LeDOM.git
   ```

2. **Ouvrir dans le navigateur**
   ```bash
   # Avec VS Code + Live Server (recommandé)
   code .
   # Puis clic droit sur LeDOM.html > "Open with Live Server"
   ```
   Ou simplement ouvrir `html/LeDOM.html` directement dans votre navigateur.

## 🧠 Ce que vous allez apprendre

- ✅ Ce qu'est le DOM et comment il est construit par le navigateur
- ✅ La notion de **nœuds** (texte, éléments, attributs)
- ✅ Sélectionner des éléments : `getElementById`, `querySelector`, `querySelectorAll`…
- ✅ Modifier le contenu et les styles dynamiquement
- ✅ Créer, insérer et supprimer des éléments en JavaScript
- ✅ Gérer les **événements** : clics, formulaires, clavier…
